# 📊 Data to Dollars - Indian Stock Analyzer

A free, open-source web tool for Indian investors to analyze stocks using the **8 Pillars framework** and **DCF valuation**.

## 🎯 Features

### Screen 1: Metrics
- Real-time stock price and changes
- Key financial metrics (P/E, ROE, ROIC, etc.)
- 5-year historical data
- Professional read-only display

### Screen 2: Financials
- Income Statement (Annual/Quarterly toggle)
- Balance Sheet with Assets, Liabilities, Equity
- Cash Flow Statement
- Indian fiscal year format (FY 2024-25)
- Numbers in Crores (₹)

### Screen 3: Stock Analyzer - 8 Pillars
- **Profitability:** Profit Margin, ROE, ROIC, EBITDA Margin
- **Growth:** Revenue Growth, Net Income Growth, ROIC Trend, FCF Growth
- **Valuation:** P/E, P/FCF, P/S, EV/EBITDA
- **Health:** Debt/Equity, Current Ratio, Interest Coverage, FCF/Debt
- **Dividends:** Yield, Payout Ratio, Growth, Years of Dividends
- **Quality:** Earnings Quality, FCF/Net Income, Asset Turnover, Accruals
- **Value:** Price to Book, PEG, FCF Yield, Earnings Yield
- **Custom:** Create your own pillar combinations

### Screen 4: DCF Valuation (Coming Soon)
- 10-year projection model
- 3 scenarios: Low, Mid, High
- Automatic intrinsic value calculation
- Margin of Safety adjustments

- **Data:** JSON (static for now)
- **Hosting:** GitHub Pages (free)
- **Performance:** ~500KB total size, loads instantly

## 🎓 Investment Framework

Built on principles from:
- **Everything Money** - 8 Pillars framework

- **Graham & Dodd** - Fundamental analysis
- **Buffett** - Margin of Safety concept

## 💡 Philosophy

**"Teach the framework, not recommend stocks"**

This tool teaches Indian investors:
1. What metrics matter
2. How to analyze companies
3. How to value stocks fairly
4. How to apply margin of safety
